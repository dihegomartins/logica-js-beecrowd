# 🚀 Resolução de Desafios: Beecrowd (JavaScript)

Este repositório contém as minhas soluções para os desafios da plataforma **Beecrowd** (antigo URI Online Judge), focando no desenvolvimento de lógica de programação sólida e algoritmos eficientes utilizando **Node.js**.

## 📌 Porquê este repositório?
Como estudante de **Engenharia de Software (5º Período)**, acredito que a base de um bom programador Frontend não é apenas o visual, mas a capacidade de resolver problemas lógicos complexos. Este treino diário prepara-me para lidar com sistemas de grande escala, como os encontrados no setor financeiro.

## 🛠️ Tecnologias e Ferramentas
- **Linguagem:** JavaScript (ES6+)
- **Ambiente:** Node.js
- **Versionamento:** Git

## 📂 Estrutura do Projeto
Os exercícios estão organizados por categorias e IDs da plataforma:
- `/Iniciante`: Problemas básicos de entrada, saída e aritmética.
- `/Ad-Hoc`: Problemas de simulação e lógica aplicada.
